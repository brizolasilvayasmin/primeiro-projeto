# Primeiro Projeto — To-Do List
Este projeto é uma aplicação de lista de tarefas desenvolvida com HTML, CSS e JavaScript puro. A proposta foi criar uma interface simples, moderna e funcional para praticar conceitos fundamentais do desenvolvimento frontend, como estruturação de páginas, estilização responsiva, manipulação do DOM e armazenamento de dados no navegador utilizando localStorage.

A aplicação permite adicionar novas tarefas, marcar tarefas como concluídas, remover itens da lista e filtrar as tarefas entre pendentes e concluídas. Além disso, os dados permanecem salvos mesmo após atualizar a página, proporcionando uma experiência mais completa e próxima de aplicações reais.

O desenvolvimento deste projeto teve como foco o aprendizado e a prática das principais bases do frontend sem a utilização de frameworks ou bibliotecas externas, utilizando apenas tecnologias nativas da web. Também foram aplicados conceitos de organização de código, responsividade e experiência do usuário.

Tecnologias utilizadas: HTML5, CSS3 e JavaScript.

Para executar o projeto, basta abrir o arquivo index.html em qualquer navegador.
